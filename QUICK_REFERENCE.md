# Quick Reference Card

## 🚀 Setup (Do This First)

```
1. Go to https://app.supabase.com
2. SQL Editor → New Query
3. Copy SUPABASE_SQL_SETUP.sql
4. Paste and Run
5. Authentication → Users → Add user
6. Email: admin@mylinks.local
7. Password: Your choice
8. Done! ✅
```

## 🔐 Login

```
URL: /admin/admin.html
Password: The one you created above
```

## ➕ Add Link

```
1. Click "Add Link"
2. Fill form
3. Click "Add Link"
4. Link appears on main website instantly!
```

## ✏️ Edit Link

```
1. Click "My Links"
2. Find link in table
3. Click ✏️ icon
4. Update form
5. Click "Save Changes"
```

## 🗑️ Delete Link

```
1. Click "My Links"
2. Find link in table
3. Click 🗑️ icon
4. Confirm deletion
```

## 📥 Export Links

```
1. Click "Settings"
2. Click "Export Links"
3. JSON file downloads
```

## 📤 Import Links

```
1. Click "Settings"
2. Click "Import Links"
3. Select JSON file
4. Links added to database
```

## 🔄 Real-Time Sync

```
Admin Panel → Supabase → Main Website
(Instant, no refresh needed)
```
