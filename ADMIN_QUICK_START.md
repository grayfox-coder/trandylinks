# Admin Panel Quick Start

## Access Admin Panel
- URL: `/admin/admin.html`
- Password: The one you set in Supabase (default: `admin123` if not changed)

## Add a Link
1. Click **Add Link** in sidebar
2. Fill in the form
3. Click **Add Link** button

## Edit a Link
1. Click **My Links** in sidebar
2. Find the link in the table
3. Click the ✏️ icon
4. Update the form
5. Click **Save Changes**

## Delete a Link
1. Click **My Links** in sidebar
2. Find the link in the table
3. Click the 🗑️ icon
4. Confirm deletion

## Export Links
1. Click **Settings** in sidebar
2. Click **Export Links** button
3. JSON file downloads to your computer

## Import Links
1. Click **Settings** in sidebar
2. Click **Import Links** button
3. Select a JSON file
4. Links are added to your database

## Clear All Links
1. Click **Settings** in sidebar
2. Click **Clear All Links** button
3. Confirm twice (this cannot be undone!)

## Change Password
1. Click **Settings** in sidebar
2. Enter new password (min 6 characters)
3. Click **Update Password**
4. Use new password next time you login

## Logout
- Click the logout button (top right of sidebar)
- You'll be returned to the login screen

## Link Categories
- **Recommended**: Your top picks
- **Deals**: Special offers
- **AI Tools**: AI-powered services
- **Hosting**: Web hosting services
- **Finance**: Financial products
- **Health**: Health & wellness
- **Resources**: Educational resources

## Link Badges
- **HOT**: 🔥 Trending/popular
- **FREE**: 🎁 Free service/trial
- **DEAL**: 🤝 Special offer

## Tips
- Use high-quality images (PNG with transparent background recommended)
- Keep descriptions short (2 lines max)
- Use clear, action-oriented CTA text
- Check the "Highlight" box for your best links
- Links appear on main website instantly after adding
